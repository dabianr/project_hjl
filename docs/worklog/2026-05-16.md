# 2026-05-16 工作日志

项目：BlockProof — 区块链电子文件存证系统

## 核心改动

### 后端
- **/logs 端点加 uploader 过滤**：支持按设备标识筛选记录。传 `uploader=xxx` 时 SQL 查 `WHERE uploader = ?`，不传则全量。limit/offset 从默认值改为 `Query(ge=1, le=100)` 校验

### 前端
- **新建 MyEvidence.jsx**：我的存证列表组件，按 device_id 过滤显示个人记录。功能：分页（每页 10 条）、文件类型图标、完整存证详情（文件名/时间/区块号/SM3/IPFS/交易哈希）、每条复制按钮、骨架屏加载态、空状态占位
- **App.js 加 tab**：导航栏新增「我的存证」tab，位于「上传存证」和「存证列表」之间。导入 MyEvidence 并用 ErrorBoundary 包裹

### 构建验证
- `npm run build` 编译通过，source map 确认 MyEvidence.jsx 打入 bundle
- 沙箱写的前端文件宿主机 inotify 不感知，需手动重启 `npm start`
