# DEBUG日志 — 2026-05-14

## 问题1: 存证记录分页翻页后回弹

**现象**: 点击下一页后分页自动跳回第一页
**根因**: `fetchLogs` 的 `useCallback` 依赖 `[page]`，`useEffect` 又依赖 `fetchLogs`。改页 → 生成新函数引用 → useEffect 重跑 → 强制回到第 0 页
**修复（v1）**: 用 `useRef` 隔离 page 状态，`useCallback` 依赖 `[]`
**结果**: 未生效，因 `useEffect` 仍重跑
**修复（v2）**: 分页逻辑交给 EvidenceList 本地 `useState` 管理，App.js 只传初始数据和 refresh
**结果**: 有效

## 问题2: 存证记录不显示（"暂无存证记录"）

**现象**: 上传存证后切到列表页看不到任何记录
**根因**: `App.js` 的 `fetchLogs` 函数残留了已删除变量 `PAGE_SIZE` 和 `setTotalLogs`，API 请求直接异常被空 catch 吞掉
**修复**: 替换 fetchLogs 为干净版本 `async () => { ... limit: 10, offset: 0 ... }`
**验证**: build 后 grep PAGE_SIZE = 0

## 问题3: 分页控件消失

**现象**: 翻页功能修复后分页按钮不显示
**根因**: `EvidenceList` 的 `total` 状态初始值 0，`Math.ceil(0/10) = 1`，分页因 `totalPages <= 1` 隐藏
**修复**: 组件挂载时独立调 API 获取总记录数

## 问题4: EvidenceList 暗色卡片为白色（发白）

**现象**: 暗色模式下卡片为白色/浅灰色
**根因**: `dark` class 挂在 `<div>` 上，CSS 选择器 `html:not(.dark)` 检查的是 `<html>` 元素，永远匹配不到，所以日间颜色覆盖了暗色
**修复**: `document.documentElement.classList.toggle("dark", isDark)`

## 问题5: Cloudflare 缓存不更新

**现象**: 代码已更新但公网访问仍为旧版
**方案A**: 去 Cloudflare 后台 Purge Everything — 用户已操作但效果不稳定
**方案B**: `npm run build` 产出带 hash 文件名 → nginx 托管 → Cloudflare 看到新 URL 强制拉新
**结果**: nginx `root + tryFiles` 在 NixOS 模块中报 500，最终回退 proxy 模式

## 问题6: nginx root + tryFiles 报 500

**现象**: NixOS nginx 模块配置 `root` + `tryFiles` 后访问返回 500
**根因**: 疑似 NixOS nginx 模块不支持 `root` 与 `tryFiles` 的搭配组合
**修复**: 回退到 proxy 模式（`proxyPass http://127.0.0.1:3000`）
