# 2026-05-18 Debug 日志

## 1. PDF 证书中文渲染黑框（ZapfDingbats）

**症状**：下载的 PDF 中所有中文显示为乱码/黑框

**排查**：
- 解压 PDF stream（ASCII85Decode + FlateDecode）
- 发现 `/F3`（ZapfDingbats）引用 13 处
- 所有中文字符串 `(nnnnnnnnnnn)` 用 ZapfDingbats 渲染

**根因**：Helvetica 等标准 Type1 字体无 CJK 字形。reportlab 找不到字形时自动 fallback 到 ZapfDingbats 符号字体

**修复**：注册 Ghostscript 自带的 DroidSansFallback.ttf 为 CJKFont，所有中文用 CJK_FONT 渲染

## 2. PDF 证书 QR 码黑框

**症状**：QR 码位置出现黑色矩形

**排查**：
- 代码用 `c.drawImage()` 从 BytesIO 画 PNG
- reportlab 不抛异常，PDF stream 无图像对象
- 画出的黑色矩形宽高与 QR 码区域一致

**根因**：Pillow 在 NixOS 上 PNG 编码/解码静默失败，reportlab 找不到图像数据时画黑矩形占位

**修复**：用 `qrcode.QRCode.modules()` 获取二维码二维布尔矩阵，用 reportlab `rect()` 逐一绘制黑白方块

## 3. 二维码扫码 525

**症状**：手机扫 PDF 二维码 → Cloudflare 525 SSL handshake failed

**排查**：
- `dig` 显示 apex（bigdogbarks.top）和 www 指向相同 Cloudflare IP
- 直接访问 `www.bigdogbarks.top` 正常
- 二维码链接到不带 `www` 的 apex

**根因**：Cloudflare 上 apex 域名的 SSL/Tunnel 配置与 www 子域名不一致

**修复**：verify_url 改为 `https://www.bigdogbarks.top/?verify=xxx` 带 `www.` 前缀

## 4. Card 发光动效诡异

**症状**：卡片 hover 时出现一个紫色光点绕卡旋转

**排查**：CSS `.card-glow::before` 用 `conic-gradient` 定义单道紫色射线 + `transform: rotate(360deg)` 动画

**根因**：conic-gradient 只有一个有色 stop，其余为 transparent。旋转动画产生单点螺旋桨效果

**修复**：删 `::before` 伪元素，hover 时用 `box-shadow` 双层紫光晕 + 边框颜色过渡
