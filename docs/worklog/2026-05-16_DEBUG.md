### 调试过程

**问题 1：npm run build 输出 hash 不变**
- 症状：连续 3 次 `npm run build` 输出 `main.acdfc677.js`，中文标签在 bundle 中搜不到
- 根因：CRA webpack 有缓存层（`node_modules/.cache/babel-loader/`），沙箱写的文件虽然存在，但 webpack 读的是层级不同的缓存。清 `node_modules/.cache` 后重建才变化
- 解决：`sudo rm -rf node_modules/.cache && sudo mkdir -p node_modules/.cache && sudo chmod 777 node_modules/.cache`

**问题 2：宿主机 npm start 报 eslint EACCES**
- 症状：用户运行 `npm start` → `[eslint] EACCES: permission denied, open '.cache/.eslintcache'`
- 根因：沙箱创建的 `.eslintcache` 文件属主 root/hermes，宿主机用户 hjl 无法写入
- 解决：`sudo chmod -R ugo+rwX node_modules/.cache`
- 预防：改前端文件后顺手跑 `sudo chmod -R ugo+rwX frontend/node_modules/.cache`
