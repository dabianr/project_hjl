# 2026-05-13 工作日志（也许是最终版）

项目：BlockProof — 区块链电子文件存证系统
状态：✅ 公网部署完成，功能正常运行

## 核心改动

### 基础设施
- 链启动改为 Foundry anvil（Hardhat SIGBUS 不可用）
- anvil 状态每 30 秒自动存盘到 `.anvil/`，重启不丢数据
- 合约地址持久化，重启复用不重新部署
- 后端端口 8001（避开 8000 nginx 占用）
- 日志移到项目 `.logs/` 目录防权限冲突

### 公网部署
- Cloudflare Tunnel + nginx 反代架构
- 前端 `/` → :3000，API `/api/*` → :8001，远程桌面 `/vnc` → :6080
- nginx `client_max_body_size 50M` 支持大文件上传
- `Cache-Control: no-store` 防 Cloudflare 缓存旧 bundle

### 前端功能
- 统计仪表盘：四色卡片（紫/青/绿/橙）+ count-up 数字动画 + 错峰入场
- ECharts 柱状图（7 天存证趋势）+ 空状态占位
- 拖拽上传：文件类型图标、真实上传进度、成功/失败状态覆盖
- 存证列表：复制按钮（clipboard + fallback）、相对时间、卡片悬浮
- 验证工具：分步验证动画、结果复制按钮
- 暗色/亮色主题切换
- Tab 下滑线指示器
- Footer 区块链网络状态指示
- Toast 通知 / 骨架屏 / 错误边界

### 智能合约
- 环形缓冲替代无限数组（MAX_RECENT=200）控制 gas 成本
- EvidenceCreated 事件溯源保证完整历史可查

### CI/CD
- GitHub Actions 自动 pytest
