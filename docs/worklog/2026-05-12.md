## 5-12 第四次会话：公网部署（nginx 反代 + Cloudflare Tunnel）

| # | 文件 | 内容 |
|---|------|------|
| 18 | nginx.nix | `/` → :3000（前端首页），`/api` → :8001（用 rewrite 剥前缀），`/vnc` → :6080（basic auth），去掉了 virtualHost 层的 basicAuthFile |
| 19 | cloudflared.nix | `bigdogbarks.top` 从直接 :3000 改为走 nginx :49514（noTLSVerify），去掉 app 子域 |
| 20 | frontend/src/App.js | API_BASE 从绝对路径 `http://127.0.0.1:8001` 改为相对路径 `/api` |

关键教训：NixOS nginx 模块 `locations."/api".proxyPass` 带尾缀 `/` 会被静默吞掉，必须用 `extraConfig` 内 `rewrite ^/api/?(.*)$ /$1 break;` 剥前缀。

结果：bigdogbarks.top → CF Tunnel → nginx → 前端 :3000 + API :8001，同域无跨域，访问正常。存证 16 条、今日 77 次上传。
